IGNITE HUBS CODING CHALLENGE

CHALLENGE ONE:

Embed an IMAGE using HTML and CSS.

The image must include:

  • The img tag.
  • The alt attribute.
  • Give the image a width and a height.
  • Using CSS, make the edges of the images round.
  • Using CSS:Make the image responsive
  • Centre The Image.
  • Using CSS: Make the image slightly transparent

  • CHALLENGE: Using CSS and the filter attribute, add a filter to the image. Use this W3Schools page to help you.

TIP: Use websites such as W3Schools and Code Pen for guidance but DEFINITLY do your own research.

CHALLENGE TWO:

Embed a VIDEO using HTML and CSS.

The video must include:

  • The video tag.
  • Give the video a width and a height.
  • Include the source attribute. Include two sources of the video: mp4 and ogg.
  • Include text which states that the browser does not support the video
  • Align the video to the right.
  • Make the video responsive

  • CHALLENGE: Add the poster attribute to add a thumbnail to the video. Use this W3Schools page to help you.

CHALLENGE THREE:

Embed AUDIO using HTML and CSS.

The audio must include:

  • The audio tag.
  • Include the source attribute. Include two sources of the audio: mp4 and ogg.
  • Include text which states that the browser does not support the audio
  • Align the audio to the centre of the screen.
  • Change the background colour of the audio.
  • Use the autoplay attribute

  • CHALLENGE: Using CSS, style either the play or mute buttons, the time or volume slider. Use this Blog page to help you.
